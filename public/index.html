OpenRouter 安全转发网关

这是一个基于 Cloudflare Workers 的多用户 OpenRouter 转发服务。它会在代理请求时抹除地区与 IP 等隐私线索，只保留应用来源、请求标识和用量信息，方便你做安全追溯与审计。

进入管理后台

安全边界

统一鉴权、RBAC、CSRF、防 SSRF、Header 白名单、日志脱敏。

多用户能力

管理员签发服务 Token、Key 池管理、封禁控制、审计与用量监控。

兼容转发

客户端使用你的平台代理地址访问 /v1/* 即可转发到 OpenRouter。

接入说明

  • 服务地址示例：https://your-domain.example/v1/chat/completions
  • 客户端的 Authorization 使用平台签发的服务 Token。
  • 上游 OpenRouter Key 由平台托管，不暴露给接入方。
  • 应用来源信息默认取自服务 Token 绑定的 app_name / app_url。
